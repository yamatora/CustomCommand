# LINE Pay analyzer

`LINE Pay`を利用するにあたり、`LINEウォレット`のトーク履歴から支出内容をまとめる

## メモ

1. 日付区切り
    
    `yyyy/MM/dd`区切りの配列を取得

2. 決済区切り

    `hh:mm`区切りの配列を取得

3. 内容取得, 出力

    - 日時
    - 加盟店: 使途として
    - 金額: ` `(半角スペース)区切りの配列を取得した時に後ろ２項目が[`1,000`, `円`]のようになる
        - `チャージ`, `お支払い`となっている項目のみ取得
